# Shared

Use `shared` para codigo reutilizavel que nao depende de uma feature especifica.

Coloque aqui:

- componentes genericos;
- directives;
- pipes;
- models reutilizaveis;
- funcoes utilitarias puras.

Evite services com estado global nesta pasta. Services globais pertencem ao `core`; services de negocio pertencem a uma feature.
